# Career Guide

Career Guide is a modern career guidance platform designed to help students explore careers, entrance exams, colleges, scholarships, and educational pathways.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- JSON
- Git
- GitHub
